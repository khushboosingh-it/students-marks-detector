# 🎓 Student Marks Detector (Python Project)

## 📌 Overview

This is a beginner-level Python project that generates random marks for a student and assigns a grade based on performance.

The program takes user input (name and seat number), generates marks using Python’s random module, and displays the final result with a grade.

## 🚀 Features

* Takes student name and seat number as input
* Generates random marks (0–100)
* Assigns grade based on marks
* Displays a formatted result

## 🛠️ Technologies Used

* Python (Core Concepts)
* Random Module

## 🧠 Concepts Used

* User Input (`input()`)
* Conditional Statements (`if-elif-else`)
* Random Number Generation (`random.randint()`)
* String Formatting (f-strings)

## 💡 How It Works

1. User enters name and seat number
2. Program generates random marks between 0–100
3. Based on marks, a grade is assigned:

   * 95+ → A++
   * 85–94 → A
   * 75–84 → B
   * 60–74 → C
   * Below 60 → Fail
4. Final result is displayed
